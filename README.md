# Festival Management System

این پروژه با هدف مدیریت دیجیتال رویدادهای فرهنگی و جشنواره‌ها توسط شهرداری طراحی شده است. سامانه شامل کلاس‌های Event، Participant و Manager بوده و با استفاده از C++، ساختار چندفایلی، استانداردهای کدنویسی، STL Containers و Exception Handling توسعه یافته است.

## امکانات سیستم
- افزودن رویداد جدید
- ثبت شرکت‌کننده جدید
- ثبت‌نام شرکت‌کننده در رویداد با بررسی ظرفیت و مدیریت خطاها
- جستجوی رویداد یا شرکت‌کننده با شناسه
- نمایش لیست رویدادها بر اساس نوع یا تعداد شرکت‌کنندگان
